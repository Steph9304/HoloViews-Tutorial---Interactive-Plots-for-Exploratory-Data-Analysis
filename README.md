# HoloViews-Tutorial---Interactive-Plots-for-Exploratory-Data-Analysis
A tutorial for using HoloViews for exploratory data analysis with a Paris housing dataset

This repository will contain 5 files, this README, the tutorial (a Jupyter Notebook with code, markdown cells, and comments walking you through the process), 3 html files (one for each of the 3 main graphs, so that you can get a closer look at each), and a pdf file for a writeup about this project.

In this tutorial, I will be showing you how to make 3 different kinds of HoloViews plots for exploratory data analysis. My example dataset is a Paris housing dataset from kaggle. The Holoviews website includes much documentation and guidance, but not geared very much toward beginners. The purpose of my project is to display information about how to make HoloViews plots in a very beginner-friendly way. I will be boiling these plots down to their simplest forms and then showing step-by-step how to improve them to be high-level plots that can help one explore and display information about a dataset.
